3. JDK 21

6. Thymeleaf

<br/>

# 게시판 주요기능
1. 글쓰기 `(/board/save)`
2. 글 목록 `(/board/)`
3. 글 조회 `(/board/{id})`
4. 글 수정 `(/board/update/{id})`
5. 글 삭제 `(/board/delete/{id})`
6. 페이징 처리 `(/board/paging)`
